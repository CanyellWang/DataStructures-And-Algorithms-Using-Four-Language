### 列表
列表是非常常见的一种数据结构，比如日常所见的购物清单、待办事项等等。它提供了对列表数据的一系列操作，比如：添加、删除、修改、遍历等功能。当我们把这样的具体问题抽象成用列表去解决的时候，往往可以简化问题。
